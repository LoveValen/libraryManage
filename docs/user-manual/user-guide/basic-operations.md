## 🔐 账户管理

### 注册新账户

#### 步骤1: 访问注册页面
1. 打开图书管理系统首页
2. 点击右上角"注册"按钮
3. 或者点击登录页面下方的"立即注册"链接

#### 步骤2: 填写注册信息
```
必填信息：
├── 用户名（3-20个字符，字母数字下划线）
├── 邮箱地址（用于接收系统通知）
├── 真实姓名（用于借阅凭证）
├── 密码（至少8位，包含字母和数字）
└── 确认密码

选填信息：
├── 手机号码（用于短信通知）
├── 个人简介
└── 头像上传
```

#### 步骤3: 验证邮箱
1. 检查注册邮箱中的验证邮件
2. 点击邮件中的验证链接
3. 返回登录页面完成验证

> 💡 **提示**: 如果没有收到验证邮件，请检查垃圾邮件文件夹，或联系管理员。

### 登录系统

#### 标准登录
1. 在登录页面输入用户名/邮箱和密码
2. 可选择"记住我"保持登录状态
3. 点击"登录"按钮

#### 忘记密码
1. 点击登录页面的"忘记密码"链接
2. 输入注册邮箱地址
3. 检查邮箱中的重置密码链接
4. 设置新密码

#### 多设备登录管理
- 查看当前登录设备：个人中心 → 安全设置 → 登录设备
- 远程登出其他设备：点击设备列表中的"登出"按钮
- 登录异常提醒：系统会发送异地登录通知

## 👤 个人资料设置

### 基本信息编辑

#### 访问个人资料
1. 点击右上角头像
2. 选择"个人中心"
3. 进入"基本信息"页面

#### 可编辑字段
```
基本信息：
├── 真实姓名 ⚠️（影响借阅证件，需管理员审核）
├── 邮箱地址 ⚠️（需要重新验证）
├── 手机号码 ⚠️（需要短信验证）
├── 个人简介 ✅（可随时修改）
├── 所在地区 ✅
└── 生日 ✅

展示设置：
├── 头像上传 ✅（支持JPG/PNG，最大5MB）
├── 个人主页背景 ✅
├── 个性签名 ✅（最多100字符）
└── 兴趣标签 ✅（最多10个标签）
```

> ⚠️ **注意**: 重要信息（姓名、邮箱、手机）的修改可能需要管理员审核。

### 头像设置

#### 上传头像
1. 点击头像区域的"上传"按钮
2. 选择图片文件（JPG/PNG格式，最大5MB）
3. 使用裁剪工具调整头像区域
4. 点击"保存"完成上传

#### 头像要求
- 文件格式：JPG、PNG
- 文件大小：最大5MB
- 推荐尺寸：200x200像素
- 内容要求：真人头像，内容健康

## 🔒 安全设置

### 密码管理

#### 修改密码
1. 进入个人中心 → 安全设置
2. 点击"修改密码"
3. 输入当前密码
4. 设置新密码（至少8位，建议包含大小写字母、数字、特殊字符）
5. 确认新密码

#### 密码强度要求
```
密码要求：
├── 长度：8-128个字符
├── 必须包含：字母和数字
├── 建议包含：大小写字母、特殊字符
├── 禁止使用：常见弱密码、个人信息
└── 定期更新：建议每3-6个月更换一次
```

### 两步验证

#### 启用两步验证
1. 安全设置 → 两步验证
2. 选择验证方式：
   - 短信验证码
   - 邮箱验证码
   - 身份验证器APP（Google Authenticator/Microsoft Authenticator）
3. 按照指引完成设置
4. 保存备用验证码

#### 使用两步验证登录
1. 输入用户名和密码
2. 系统提示输入验证码
3. 获取并输入6位验证码
4. 完成登录

### 登录历史

#### 查看登录记录
- 位置：个人中心 → 安全设置 → 登录历史
- 信息包括：登录时间、IP地址、设备信息、地理位置
- 异常登录会特别标记

#### 安全提醒
- 异地登录会发送邮件/短信通知
- 多次登录失败会暂时锁定账户
- 发现异常活动请立即修改密码并联系管理员

## 🔔 通知管理

### 通知中心

#### 查看通知
1. 点击顶部导航栏的通知图标
2. 查看最新通知列表
3. 点击具体通知查看详情
4. 点击"查看全部"进入通知中心

#### 通知分类
```
通知类型：
├── 🔴 紧急通知（红色图标）
│   ├── 账户安全警告
│   ├── 系统故障通知
│   └── 重要政策变更
├── 🟡 重要提醒（黄色图标）
│   ├── 图书到期提醒
│   ├── 预约图书可借
│   └── 系统维护预告
├── 🔵 一般消息（蓝色图标）
│   ├── 新书推荐
│   ├── 活动通知
│   └── 功能更新
└── 🟢 社交互动（绿色图标）
    ├── 评论回复
    ├── 好友动态
    └── 点赞通知
```

#### 通知操作
- **标记已读**: 点击通知或点击"标记已读"按钮
- **批量操作**: 选中多个通知后进行批量标记/删除
- **过滤筛选**: 按类型、时间、已读状态筛选
- **搜索通知**: 在通知历史中搜索特定内容

### 通知设置优化

#### 减少通知干扰
1. 设置免打扰时段（如：22:00-08:00）
2. 选择重要通知类型（只接收紧急和重要提醒）
3. 调整推送频率（改为每日汇总）
4. 关闭不需要的社交通知

#### 重要通知确保送达
1. 启用多渠道通知（邮件+站内）
2. 设置紧急联系方式
3. 定期检查通知设置
4. 更新联系信息

## 💡 使用技巧

### 效率提升技巧

1. **善用搜索**: 掌握高级搜索语法，快速找到所需图书
2. **个性化首页**: 设置常用功能快捷方式

1. **密码安全**: 避免使用简单密码或与其他网站相同密码
2. **信息完整性**: 保持个人信息准确和及时更新
3. **隐私保护**: 合理设置隐私选项保护个人信息
4. **及时处理**: 及时处理到期图书和系统通知

## 🆘 问题解决

### 常见问题

#### 登录问题
- **忘记密码**: 使用"忘记密码"功能重置
- **账户被锁**: 联系管理员解锁
- **验证码问题**: 检查邮箱/短信接收设置

#### 功能异常
- **页面加载慢**: 检查网络连接，清除浏览器缓存
- **功能无响应**: 刷新页面或重新登录
- **数据不同步**: 手动触发同步或联系技术支持

### 获取帮助

- **在线客服**: 工作时间内可使用实时聊天功能
- **帮助中心**: 查看详细FAQ和操作指南
