## 🚀 快速开始

### 系统访问

1. **Web端访问**: [https://library.yourdomain.com](https://library.yourdomain.com)
2. **移动端**: 下载"智慧图书馆"APP或访问PWA版本
3. **管理端**: [https://admin.library.yourdomain.com](https://admin.library.yourdomain.com)

### 账户注册

1. 点击首页"注册"按钮
2. 填写基本信息（用户名、邮箱、真实姓名）
3. 设置安全密码（至少8位，包含字母和数字）
4. 验证邮箱地址
5. 完善个人资料

### 首次登录

1. 使用用户名/邮箱和密码登录
2. 阅读并同意使用条款
3. 设置个人偏好（语言、通知方式等）
4. 完成新手引导
